# React Starter API
Backend API for the React Starter project.

## Technologies

* Node
* Yarn

## Yarn
Add dependency
```
yarn add <dependency>
```

Add dev dependency
```
yarn add -D <dependency>
```

Remove dependency
```
yarn remove <dependency>
```
